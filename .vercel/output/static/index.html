WildriftGuides

Wild Rift Guides website was created using Astro, web framework optimized for building fast, content-driven websites. All content like champion data, item builds, and runes is loaded from structured JSON files and rendered at build time. Styling is done with CSS and Tailwind CSS for a clean, responsive layout across devices. The site offers champion guides with skill orders, builds, and runes, plus tier lists and blog posts on game updates. Markdown is used for articles, and interactive features like video previews and filtering are powered by light JavaScript. It’s designed to be a fast, reliable resource for Wild Rift players.
